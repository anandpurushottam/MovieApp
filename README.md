# MovieApp
A Movie app using Mvp Pattern
------------------------
#### Fetures implemented:-

* Support for tablets / mobile
* View Popular Movies
* View details of popular movies
* Watch trailer
* Add to Fav

#### Library used:-

* RxJava and rxAndroid
* Timber
* Retrofit
* Okhttp
* Gson
* Picaso
* Room

#### Resource used :-
